IPL Auction Dataset Used to Practice SQL 

This dataset contains the following details:
Player Name,
Price paid to the Player,
Type- Indian or Overseas,
Acquisition	(Auctioned/Retained),
Role	Batter/All-rounder/Bowler etc,
Team Name


The dataset was explored to find few details like :
- Total team spending
- Top two highest paid players
- Players paid higher than average Team spending
- Highest paid player of each team
- Categorization of players on amount spent etc..
